🔧 Troubleshooting

Desktop icon didn't appear?

  • Make sure you clicked "Install" in the browser dialog (not just "Cancel" or X)
  • Wait 5 seconds - icon creation may be delayed
  • Check Start Menu: Windows Key → Type "RADAI"

Install button not working?

  • Try browser menu: Click ⋮ (Chrome/Edge) → "Install RADAI"
  • Open browser console (F12) to see diagnostic logs
  • Use Chrome or Edge for best compatibility
